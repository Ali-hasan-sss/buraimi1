# Cairo Font Files

Place Cairo `.woff2` files here so `next/font/local` can load them without network access.

Required files:
- Cairo-Regular.woff2
- Cairo-Bold.woff2
- Cairo-Light.woff2
- Cairo-Medium.woff2
- Cairo-SemiBold.woff2
- Cairo-ExtraLight.woff2
- Cairo-ExtraBold.woff2
- Cairo-Black.woff2

You can download these from:
- Google Fonts: https://fonts.google.com/specimen/Cairo
- Font Squirrel: https://www.fontsquirrel.com/fonts/cairo

After adding the files, restart `npm run dev` and Cairo will be used globally.
